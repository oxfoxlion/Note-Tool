# Work Log - 2026-03-25

## Scope

本次主要聚焦在 `Space` 層導入、卡片與白板跨 Space 複製、以及側邊欄改為 `Space > Card Box / Board folders` 的導覽結構。

## Completed

1. 導入 Space 資料層與 migration
- 後端新增 `note_tool.spaces`，並將既有 cards / boards / board_folders 全部歸入每位使用者的 default space。
- 新增 migration：
  - `LineLanguageBot/scripts/migrations/007_add_note_tool_spaces.sql`
- migration 已成功套用至資料庫。

2. 後端 Space API 與 default space 邏輯
- 新增 / 擴充：
  - `LineLanguageBot/services/note_tool/note_tool_space.js`
  - `LineLanguageBot/routes/note_tool/space_routes.js`
- 支援：
  - `GET /note_tool/space`
  - `GET /note_tool/space/:spaceId`
  - `POST /note_tool/space`
  - `PUT /note_tool/space/:spaceId`
  - `DELETE /note_tool/space/:spaceId`
- 規則：
  - default space 不可刪除
  - 刪除非 default space 時，該 space 下資料會一併刪除

3. cards / boards / folders 改為支援 `space_id`
- 後端 cards / boards / folders 查詢與建立流程已改為接受 `space_id`。
- 若前端未帶 `space_id`，仍可 fallback 到該使用者 default space。
- 前端 API 更新：
  - `lib/noteToolApi.ts`
- 主要能力：
  - `getCards(spaceId?)`
  - `getBoards(folderId?, spaceId?)`
  - `getBoardFolders(spaceId?)`
  - `createCard({ ..., space_id })`
  - `createBoard({ ..., space_id })`
  - `createBoardFolder({ name, space_id })`

4. 卡片跨 Space 複製
- 後端新增：
  - `POST /note_tool/card/:cardId/copy-to-space`
- 前端新增 `Copy to space`，只接在既有卡片 `...` menu：
  - `app/(app)/cards/[id]/CardDetailClient.tsx`
  - `components/CardOverlay.tsx`
  - `components/cards/CardCopyToSpaceModal.tsx`

5. 白板跨 Space 複製
- 後端新增：
  - `POST /note_tool/board/:boardId/copy-to-space`
- 複製內容包含：
  - board 本體
  - board regions
  - board 上的 card 配置
  - 需要時一併複製卡片到目標 Space
  - 重新對應複製後卡片之間的 internal links
- 前端新增 `Copy to space`，只接在既有白板 `...` menu：
  - `app/(app)/boards/page.tsx`
  - `app/(app)/boards/[id]/page.tsx`
  - `components/boards/BoardCopyToSpaceModal.tsx`

6. 還原既有卡片 layout，避免功能改動影響外觀
- 依需求將卡片列表、卡片本體、modal、sidebar 的非必要 layout 改動還原。
- `Copy to space` 僅保留在既有 `...` actions menu，不另開新按鈕入口。

7. 側邊欄改為 Space 分組導覽
- `app/(app)/layout.tsx` 改為：
  - `Space`
    - `Card Box`
    - 該 Space 的 board folders
- 行為：
  - 點 Space 名稱切到該 Space 的 boards
  - 點 `Card Box` 切到該 Space 的 cards
  - 點 folder 切到該 Space 的 boards folder
  - 目前 Space 會保持展開，其它 Space 可手動展開 / 收合

8. Space `...` menu
- 每個 Space 標題列新增 `...` menu，提供：
  - `Create folder`
  - `Add space below`
  - `Delete space`
- `Add space below` 會把新 Space 插到目前 Space 下方。
- Space 順序目前以前端 localStorage 保存。

## Behavior Changes

1. 側邊欄不再是單一層 `Card Box / Boards / folders`，而是依 Space 分段顯示。
2. 卡片與白板都可複製到其他 Space，但不能直接移動。
3. Space 與 board folders 的 local state 已改成 per-space，不再共用同一份 folder order。
